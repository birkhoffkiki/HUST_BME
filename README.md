# HUST_BME_DSP
此类主要为华科BME的DSP实验所设计。主要有两部分组成：
1.Matrix 类：基本的数据类型，是本程序设计的核心。
2.mat 类：提供了多种对矩阵的操作方法，具体详见mat.h
